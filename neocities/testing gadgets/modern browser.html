Welcome

Hello! This is a HTML/CSS theme created by 5493 Arts! This theme is free to use and modify to fit your needs; just please don't remove the tooltip credit (hovering the menu icon)!

I have annotated the code in this to explain certain things to better your understanding of how it is set up. I would highly recommend that you customize this theme to your liking; to learn more about HTML/CSS, please refer to a source like W3 Schools or MDN Web Docs.

This theme is built with grids. If you are unfamiliar with grids, please check out a source like W3 Schools explanation of grid layouts.

This theme uses Inter, a Google Font that is 100% free to use. The stylesheet utilizes the Google Fonts CSS API, so you do not have to download the font to use it! This layout also utilizes the svg icons available on Google Fonts, therefore the icons work the same way.

This theme is not mobile responsive. Due to the way the decorative icons are positioned, it would be difficult to make it mobile responsive with pure html/css. Therefore, I suggest that you urge the viewers of your site to view it on a desktop device.
